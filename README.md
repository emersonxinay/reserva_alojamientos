# Para crear en pdf las tablas en diagrama de la base de datos
En gemfile
```gemifile 
gem 'rails-erd'
```
Para instalar 
```bash
bundle install
```
Para generar el pdf en diagrama con la base de datos
```bash
bundle exec erd
```
